# Laboratorio Api Rest

Laboratorio de API REST con persistencia en base de datos relacional SQL Server.

## Instrucciones

Existen dos modelos:

 - Author
    - Id (Type int)
    - Name (Type string)
    - LastName (Type string)
    - Birth (Type string)
    - CountryCode (Type string)

 - Book
    - Id (Type int)
    - Title (Type string)
    - Published (Type DateTime)
    - Description (Type string)
    - AuthorId (Type int)
